Understanding CSS


CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be dispalyed on screen,paper or in other media, CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

Ways we can apply CSS

  • Style tag
    • One way to include CSS into your HTML is style tags which do require a closing tag. This is referred as internal styling. The style tags allows you to type CSS between them. These can be put anywhere on your page but it's best practice to put them above your content or in the head tag.

  • Link tag
    • The link tag is the most common way to include an entire CSS file into your HTML page. This is callede using an external stylesheet. This tag does not require a closing tag. This tag should be put in the tags on your page.

  • Inline Style Attribute
    • You can include a style argument on most HTML tags. This allows you to style that individual tag similar to an id. Syntax for inline is
    style="property:value;property:valuel;property:value;"
